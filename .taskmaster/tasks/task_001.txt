# Details:
使用前端框架（如Electron with React/Vue）搭建应用骨架。
1. 创建主进程文件 `main.js` 和渲染进程文件 `renderer.js`。
2. 设计主应用窗口，尺寸可调，并设置标题为 "TaskMaster UI配置工具"。
3. 实现一个多标签页布局，包含 "服务商管理"、"模型配置"、"配置管理" 和 "日志查看" 四个主要标签页。
4. 每个标签页暂时使用占位符内容，确保切换逻辑正常工作。

**伪代码:**
```javascript
// main.js
app.on('ready', () => {
  const mainWindow = new BrowserWindow({ width: 800, height: 600 });
  mainWindow.loadFile('index.html');
});

// renderer.js
setupTabs(['服务商管理', '模型配置', '配置管理', '日志查看']);
```

# Test Strategy:
启动应用，验证主窗口是否正确显示。点击不同标签页，检查内容区域是否相应切换，且无视觉错误。
