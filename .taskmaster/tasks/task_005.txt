# Task ID: 5
# Title: 模型配置模块开发
# Status: done
# Dependencies: 4
# Priority: medium
# Description: 实现模型的添加、编辑和删除功能，允许用户管理与特定服务商关联的模型及其参数。
# Details:
在 "模型配置" 标签页中实现模型管理功能。
1.  **UI设计**: 使用下拉菜单让用户选择一个服务商。下方是一个列表，显示该服务商下已配置的模型。
2.  **添加模型**: 提供 "添加模型" 功能，弹出一个表单，让用户从该服务商已加载的模型ID列表中选择一个模型，并配置其特定参数（如别名、温度、top_p等）。
3.  **编辑与删除**: 列表中每一项都应有关联的 "编辑" 和 "删除" 按钮，功能类似于服务商管理。
4.  **数据结构**: 模型数据应作为服务商配置下的一个子对象数组进行存储。

# Test Strategy:
选择一个服务商，添加一个新模型配置，保存后验证数据是否正确写入配置文件。编辑该模型的参数（如温度），保存并验证更新。删除该模型，验证其是否从配置文件中移除。
